# Solana Development Kit

The **Solana Development Kit** (`@connectorkit/sdk`) is a comprehensive toolkit for building Solana applications. It provides everything from low-level RPC management to high-level React hooks, designed for both beginners and enterprise developers.

## Bundle Sizes & Import Strategies

The SDK offers multiple entry points for different use cases:

```typescript
// Full SDK (90KB) - Everything included
import { useBalance, createClient } from '@connectorkit/sdk'

// React-only (70KB) - React hooks + providers  
import { useBalance } from '@connectorkit/sdk/react'

// Client-only (30KB) - Server-side, no React
import { createClient } from '@connectorkit/sdk/client'

// Core/Minimal (15KB) - Essential hooks only (Future)
import { useBalance } from '@connectorkit/sdk/core'
```

## Quick Start

### React Applications

```tsx
import { ArcProvider, useBalance, useWalletAddress } from '@connectorkit/sdk'

function App() {
  return (
    <ArcProvider>
      <WalletBalance />
    </ArcProvider>
  )
}

function WalletBalance() {
  const { address } = useWalletAddress()
  const { balance, isLoading } = useBalance({ address })
  
  if (isLoading) return <div>Loading...</div>
  
  return <div>Balance: {balance} SOL</div>
}
```

### Server-Side Usage

```typescript
import { createClient } from '@connectorkit/sdk/client'

const client = createClient({
  rpcUrl: 'https://api.mainnet-beta.solana.com'
})

const balance = await client.getBalance('4fYNw3dojWmQ4dUQFNUTcgd6fj6vKWUvkEvPEqKK5dL')
console.log(`Balance: ${balance} lamports`)
```
